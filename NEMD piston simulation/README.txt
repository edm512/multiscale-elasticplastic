Description of files and folders
--------------------------------
FILES
--------------------------------
run_ramp.py - main script to run NEMD piston simulation. 
Cu_mishin1.eam.alloy - interatomic potential used for copper.
in.init_long - initialisation section of LAMMPS script.
in.equil - equilibration section of LAMMPS script.
in.regions_groups - define regions and groups section of LAMMPS script.
in.set_comp_var - define system-wide computes and variables section of LAMMPS script.
in.group_info - define computes and variables for groups section of LAMMPS script. 
in.output - define output section of LAMMPS script. 
in.run_sim_long - Add voids and run simulation section of LAMMPS script. 

--------------------------------
FOLDERS
--------------------------------
ramp_profiles - folder for ramp compression profiles to define velocity of piston.
